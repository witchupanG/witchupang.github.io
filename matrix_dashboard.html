1. ภาพรวมระบบ
Room Height Matrix Dashboard คือเครื่องมือบริหารจัดการข้อมูลห้องและ Ceiling Height พร้อม Series Name แสดงผลในรูปแบบตาราง Matrix ที่ใช้งานง่าย รองรับ Import/Export Excel และมีกราฟวิเคราะห์ข้อมูลในตัว
องค์ประกอบหลัก
  • Stat Cards — แสดงสรุปจำนวนห้อง รายการ Series และ Height ที่ใช้งาน
  • Filter Bar — กรองด้วย Room, Height, Series แบบ multi-select พร้อม Search textbox
  • Matrix Table — ตาราง Room × Height พร้อม Series ในแต่ละ Cell รองรับ Drag & Drop
  • Charts — กราฟวิเคราะห์ 3 รูปแบบ อัปเดตตาม filter อัตโนมัติ
  • Import / Export — รองรับ .xlsx ในรูปแบบเดิม
2. การเริ่มต้นใช้งาน
เปิดไฟล์
  1. ดับเบิลคลิกไฟล์ matrix_dashboard.html ด้วยเบราว์เซอร์ (Chrome, Edge, Firefox)
  2. รอให้ Library โหลดจาก CDN — ต้องการอินเทอร์เน็ตครั้งแรก
  3. ระบบแสดงข้อมูลตัวอย่าง Room A / Room B ให้เห็นภาพรวม
หมายเหตุ: ข้อมูลเก็บใน RAM เท่านั้น ปิดเบราว์เซอร์แล้วข้อมูลหาย — Export ก่อนออกเสมอ
นำเข้าข้อมูลจาก Excel
กดปุ่ม Import .xlsx เพื่อโหลดข้อมูลจากไฟล์ที่มีอยู่
SheetColumnsหมายเหตุ
MatrixID, Room Nameข้อมูลห้องทั้งหมด
60012 MatrixDataMatrixId, CeilingHeight, Series Nameข้อมูล Series ทั้งหมด
หมายเหตุ: ระบบอ่านค่า String และตัวเลขได้ทั้งสองแบบโดยอัตโนมัติ
3. การจัดการห้อง (Room)
เพิ่มห้องใหม่
  1. กดปุ่ม + Room ในแถบ Filter
  2. พิมพ์ชื่อห้องในช่อง Room Name
  3. กด บันทึก
แก้ไขหรือลบห้อง
กดไอคอน ✎ ด้านขวาของชื่อห้องในตาราง เพื่อเปิดหน้าต่างแก้ไข
  • เปลี่ยนชื่อ แล้วกด บันทึก
  • กดปุ่ม ลบ เพื่อลบห้องพร้อม Series ทั้งหมดในห้องนั้น
ข้อควรระวัง: การลบห้องจะลบ Series ทุกตัวในห้องนั้นไปด้วยโดยไม่สามารถ Undo ได้
4. การจัดการ Series Entry
เพิ่ม Series ใน Cell
มี 2 วิธี:
  • + กดปุ่ม + ภายใน Cell — เปิด modal พร้อม Room และ Height ที่เลือกไว้แล้ว
  • + Entry กดปุ่มในแถบ Filter — เปิด modal ว่างให้เลือก Room, Height, Series เอง
การตรวจซ้ำ: ระบบตรวจสอบ Series ซ้ำใน Cell เดียวกันอัตโนมัติ หากซ้ำจะแสดง Toast แจ้งเตือนและไม่บันทึก
แก้ไข Series
คลิกซ้ายที่ Series Tag เพื่อเปิด modal แก้ไข Room, Height, หรือชื่อ Series ได้
ลบ Series
กดปุ่ม × ข้างขวาของ Series Tag เพื่อลบทันที
5. Drag & Drop
ลาก Series Tag จาก Cell หนึ่งไปวางยัง Cell อื่นได้โดยตรง
  1. คลิกค้างที่ Series Tag — cursor เปลี่ยนเป็นรูปมือจับ
  2. ลากไปยัง Cell ปลายทาง — Cell จะ highlight เส้นประสีทอง
  3. ปล่อยเมาส์เพื่อย้าย Series
หมายเหตุ: หาก Cell ปลายทางมี Series นั้นอยู่แล้ว ระบบจะแสดง Toast แจ้งว่าซ้ำและไม่ย้าย
6. Context Menu (คลิกขวา)
คลิกขวาที่ Series Tag หรือ Cell เพื่อเปิดเมนูตัวเลือก
เมนูเมื่อคลิกขวาที่ Series Tag
  • Copy series นี้ — copy เฉพาะ series ที่คลิก
  • Copy ทั้ง cell (N) — copy ทุก series ใน cell พร้อมกัน (แสดงเมื่อมีมากกว่า 1)
  • Paste — วาง series จาก Buffer ใน cell นี้
  • แก้ไข — เปิด modal แก้ไขข้อมูล
  • ลบ series นี้ — ลบทันที
เมนูเมื่อคลิกขวาที่ Cell ว่าง
  • Copy ทั้ง Cell (ถ้ามีข้อมูล)
  • Paste series จาก Buffer
  • + เพิ่ม Series ใน Cell นี้
Paste ซ้ำ: ระบบข้ามเฉพาะ series ที่ซ้ำและแสดง Toast ว่า paste ได้กี่ตัว ข้ามกี่ตัว
7. การกรองข้อมูล (Filter)
Search Textbox
  • Search Room — พิมพ์ชื่อห้องกรอง Row ในตารางแบบ Real-time
  • Search Series — พิมพ์ชื่อ Series กรอง Entry ที่ตรง
Multi-select Dropdown (Room, Height, Series)
  1. กดปุ่ม dropdown เพื่อเปิดรายการ
  2. Tick checkbox ที่ต้องการ — กด "เลือกทั้งหมด" หรือ "ล้าง" ได้
  3. กด ยืนยัน เพื่อ apply filter
Filter Height แบบช่วง (Range)
ใน dropdown Height มีแท็บ "ช่วง" ให้ตั้งค่าเริ่มต้นและสิ้นสุด เช่น 2.5m–4.0m จะแสดงเฉพาะ column ในช่วงนั้น
ล้าง Filter ทั้งหมด
กดปุ่ม ล้าง filter เพื่อ reset ทุก filter รวมถึง Search Textbox กลับสู่ค่าเริ่มต้น
8. กราฟแสดงผล
กราฟอัปเดตอัตโนมัติตาม Filter ที่ตั้งไว้ กดปุ่มแท็บเพื่อสลับโหมด
3 โหมดกราฟ
Series ต่อ HeightBar chart แนวตั้ง แสดงจำนวน Series ในแต่ละค่า Ceiling Height Series ต่อ RoomBar chart แนวนอน เปรียบเทียบจำนวน Series ในแต่ละห้อง HeatmapGrouped bar chart แสดงความสัมพันธ์ Room × Height พร้อมกัน แต่ละสีแทนห้องหนึ่ง
9. Import / Export
Import .xlsx
  1. กดปุ่ม Import .xlsx
  2. เลือกไฟล์ Excel ที่มีโครงสร้าง 2 sheets
  3. ระบบโหลดข้อมูลและแสดง Toast แจ้งจำนวนห้องและ entries ที่นำเข้า
Export .xlsx
  1. กดปุ่ม Export .xlsx
  2. ไฟล์ Matrix.xlsx จะดาวน์โหลดทันที
SheetColumnsหมายเหตุ
MatrixID, Room Nameข้อมูลห้องทั้งหมด
60012 MatrixDataMatrixId, CeilingHeight, Series Nameข้อมูล Series ทั้งหมด
หมายเหตุ: รูปแบบตรงกับไฟล์ต้นฉบับ สามารถนำเข้ากลับได้ทันที
10. เคล็ดลับและข้อควรระวัง
เคล็ดลับ
  • ใช้ Search textbox ร่วมกับ Multi-select dropdown เพื่อ filter รวดเร็วขึ้น
  • Copy ทั้ง cell แล้ว Paste ใน Room หรือ Height อื่น ประหยัดเวลากรอกซ้ำ
  • กราฟ Heatmap ดูได้ว่า Cell ไหนยังไม่มีข้อมูล
  • Export ก่อนออกจากระบบเสมอ เพราะข้อมูลไม่ Save อัตโนมัติ
  • กด Escape หรือคลิกนอก popup เพื่อปิดคู่มือได้ทันที
ข้อควรระวัง
  • ต้องการอินเทอร์เน็ตสำหรับโหลด Library (xlsx.js, Chart.js)
  • ข้อมูลเก็บใน RAM เท่านั้น ปิดเบราว์เซอร์แล้วข้อมูลหาย ต้อง Export ก่อน
  • Series ห้ามซ้ำใน Cell เดียวกัน แต่ Cell ต่างกันสามารถมีชื่อเดียวกันได้
  • การลบห้องจะลบ Series ทั้งหมดในห้องนั้นด้วย ไม่สามารถ Undo ได้
